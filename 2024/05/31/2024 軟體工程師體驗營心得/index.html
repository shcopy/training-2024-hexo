2024 軟體工程師體驗營心得

發表於 2024-05-31

去年，我有幸參加了網頁切版直播班，今年則報名了軟體工程師體驗營。這次的體驗營不僅是對去年的複習，更是一次全新的挑戰。我選擇使用 BootStrap 框架來實作 “切版任務作業二-個人品牌網站”。有了去年的學習經驗，這次寫起來顯得更加順手。
從零開始構建網頁直到完成，這種成就感是無法言喻的。

在過去，我的學習方式總是比較混亂，六角學院的系統化課程讓我受益良多。特別感謝校長和助教們的辛勤付出，不分日夜熬夜批改作業，真的非常辛苦。

這次學到的技術包括：

  • CSS 偽元素：透過學習偽元素，我能夠更靈活地控制頁面的樣式，實現一些精妙的效果。
  • SCSS 設計模式：學習了多種設計模式後，我在寫樣式時更加得心應手，代碼也變得更具結構性。
  • SCSS 變數設計：使用 SCSS 變數，使得樣式管理更加方便，尤其是在需要統一修改顏色或字體時，變數的優勢尤為明顯。
  • Hexo 部落格架站：這是我第一次使用 Hexo 架設部落格，從中學會了如何快速搭建和部署個人部落格網站。

總之，這次的軟體工程師體驗營，不僅提升了我的技術能力，還讓我對未來的學習和職業發展充滿信心。再次感謝六角學院提供這麼優質的學習平台，感謝校長和助教們的無私奉獻。期待未來能夠學到更多的知識，迎接更多的挑戰。
